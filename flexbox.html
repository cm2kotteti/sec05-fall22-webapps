We have a parent element with class flex-container holding children items. Div is used to break up an HTML document into divisions, but is a common target to hold other things that can be styled with CSS. In this case we have 3 items in our container. As the width is decreased a point will be reached where the 3 items are stacked vertically instead of horizontally. FlexBox is primarily used to linearly lay out items.

Direct child elements(s) of the flexible container automatically becomes flexible items.
